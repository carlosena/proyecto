Olvido de Clave

En caso de olvido de contraseña, el usuario podrá hacer uso del módulo de recuperación de contraseña que está disponible en el módulo de “Iniciar Sesión”, acceder al link “¿Olvidó su contraseña?” y luego siga las instrucciones.

Cambio de Clave

El usuario podrá cambiar su contraseña si lo desea por una nueva, deberá suministrar la contraseña anterior para poder realizar el proceso.

Módulo Familiar

Listar:

Los botones “perfil familiar y “lista familiar nos permiten ver una tabla con los datos de los pacientes.

Crear:

El botón crear nos lleva a un formulario que al ser diligenciado correctamente con la información y accionar el botón “Registrar”, nos permitirá crear un nuevo familiar en la base de datos.

Editar:

El botón editar nos permitirá realizar cambios puntuales en los datos de cada familiar registrado. Se mostrará una ventana tipo modal para verificar si está segur@ de continuar con el procedimiento.

Borrar:

El botón borrar nos permitirá eliminar el registro de la base de datos (por ahora solo queda inhabilitado el registro). Se mostrará una ventana tipo modal para verificar si está segur@ de continuar con el procedimiento.

El botón crear nos lleva a un formulario que al ser diligenciado correctamente con la información y accionar el botón “Registrar”, nos permitirá crear un nuevo medicamento en la base de datos.

Editar:

El botón editar nos permitirá realizar cambios puntuales en los datos de cada medicamento registrado. Se mostrará una ventana tipo modal para verificar si está segur@ de continuar con el procedimiento.

Borrar:

El botón borrar nos permitirá eliminar el registro de la base de datos (por ahora solo queda inhabilitado el registro). Se mostrará una ventana tipo modal para verificar si está segur@ de continuar con el procedimiento.

Sumar:

El botón sumar nos permite agregar nuevas existencias del medicamento indicado, esto afectará positivamente la cantidad “Stock” en la base de datos.

Suministrar medicamento:

Estos botones e indicativos nos permiten llevar los tiempos en que se va administrando un medicamento específico a un paciente puntual.

- Pendiente: Significa que todavía tiene pendiente dosis por administrar.

- Completado: Significa que ya se administraron las dosis correspondientes del día.

- Inactivo: Significa que el medicamento esta eliminado o ha sido suspendido temporalmente.

Al administrar el medicamento aparecerá una ventana modal con los datos puntuales del mismo y del paciente, así como la dosis, cantidad total diaria y stock, de forma que la persona que administra puede estar segur@ del procedimiento.

Los valores se vuelven a reiniciar cada vez que cambia la fecha del sistema.

Para listar los medicamentos que pertenecen a un solo paciente basta con buscar el nombre del paciente en el campo de “Buscar” de la tabla.

Exportar archivos

Es posible exportar en archivo digital los valores de cualquier tabla de los módulos, los botones permiten:

a. Copiar el contenido para pegarlo en otro documento.

b. Exportar a archivo con formato Excel.

c. Exportar a archivo con formato PDF.

d. Imprimir en papel.
